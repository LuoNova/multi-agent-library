# 项目经历

## 多智能体协作的高校图书馆资源调度系统

**项目时间**：2025年3月 - 至今  
**项目角色**：后端开发工程师  
**技术栈**：Spring Boot 3.5、JADE多智能体平台、MyBatis-Plus、MySQL、Vue.js、RabbitMQ

### 项目描述

该项目是一个基于多智能体协作的高校图书馆资源调度系统，旨在解决高校图书馆资源分布不均、利用率低、人工调度效率低等问题。系统通过四类智能体（用户需求感知智能体、书籍库存智能体、座位预约智能体、故障报修智能体）的协作，实现了图书资源的智能借阅与跨馆调配、座位资源的动态分配、故障资源的自动识别与联动等功能，有效提升了图书馆资源的利用效率与用户服务体验。

### 核心职责

1. **系统架构设计与实现**
   - 设计并实现了四层架构系统（视图层、智能体层、业务层、数据层），采用前后端分离模式
   - 基于JADE多智能体平台实现智能体协作机制，采用合同网协议（Contract Net Protocol）进行任务分配，使用FIPA-ACL标准语言进行消息传递
   - 设计并实现了四类智能体的协作流程，实现了智能借书、跨馆调拨、座位动态分配等核心功能

2. **图书跨馆调配模块开发**
   - 设计并实现了智能借书功能，支持本地借阅、跨馆调拨、预约排队三种策略，通过多智能体协作自动选择最优策略
   - 实现了还书自动分配功能，支持就地预留和跨馆调拨两种模式，实现了"还书-预约兑现"的自动化流程
   - 设计了投标评分公式（score = 80 - distance×10 - pending_reservation_count×5），优化跨馆调拨的决策效率
   - 实现了调拨完成回调功能，与物流系统对接，实现了调拨流程的闭环管理

3. **座位动态分配模块开发**
   - 设计并实现了座位可用性查询与动态分配功能，支持查询模式和自动选座模式，实现了座位的智能推荐
   - 实现了座位预约功能，支持预约创建、签到、暂离、结束使用等完整流程，设计了签到时间窗口和暂离时长规则
   - 实现了与借书流程的联动功能，在借书成功后自动推荐座位，实现"借书-入座"的一站式服务
   - 设计了座位排序规则（有电源优先 → 楼层升序 → 区域名升序 → 座位号升序），优化座位推荐效果

4. **故障报修模块开发**
   - 设计并实现了故障报修功能，支持用户和管理员提交故障报修，实现了故障工单的自动化处理
   - 设计并实现了资源健康查询功能，供其他智能体调用，确保调度决策考虑资源健康状况
   - 实现了故障资源联动机制，当故障发生时自动通知相关智能体调整调度策略，避免将用户分配到故障资源
   - 设计了故障工单状态流转机制（已报修 → 已受理 → 维修中 → 已恢复/已关闭），实现了故障处理的闭环管理

5. **数据库设计与优化**
   - 设计了15张核心数据表，包括用户表、书目表、副本表、借阅记录表、调拨记录表、座位表、预约表、故障工单表等
   - 通过合理的表结构设计与索引优化保证数据的一致性与查询性能
   - 实现了数据库事务管理，保证业务操作的原子性和一致性
   - 设计了数据备份策略，实现了全量备份和增量备份，保证数据可恢复性

6. **性能优化与监控**
   - 通过数据库连接池、线程池、缓存机制等技术手段提高系统性能
   - 实现了系统监控功能，记录详细的操作日志和错误日志，便于运维人员快速定位和解决问题
   - 设计了定时任务机制，实现了预留超期自动释放、预约超期自动取消、调拨延迟检测等自动化管理功能

### 项目成果

- 成功实现了图书跨馆调配、座位动态分配、故障报修三大核心模块，支撑全校师生的日常借阅和座位预约需求
- 系统支持每秒至少100次借书请求、200次座位查询请求、150次座位预约请求的并发处理能力
- 通过多智能体协作机制，实现了智能借书、跨馆调拨、座位动态分配等功能的自动化，大幅提升了图书馆资源的利用效率
- 系统可用性达到99.5%以上，全年停机时间不超过43.8小时
- 用户满意度显著提升，借书等待时间缩短50%，座位预约成功率提升30%

### 技术亮点

- **多智能体协作**：基于JADE平台实现多智能体协作，采用合同网协议进行任务分配，采用FIPA-ACL标准语言进行消息传递，实现了灵活、高效的业务协调
- **智能调度策略**：设计了三种借阅策略（本地借阅、跨馆调拨、预约排队）和投标评分公式，实现了资源调度的智能化和最优化
- **资源联动机制**：实现了故障资源与调度策略的联动，确保调度决策考虑资源健康状况，避免将用户分配到故障资源
- **业务流程自动化**：通过定时任务和智能体协作，实现了预留超期自动释放、预约超期自动取消、调拨延迟检测等自动化管理功能
