### 调拨表字段补充 (tb_book_transfer)

在原有调拨表基础上，新增以下字段：

| 字段名               | 类型        | 说明                                         |
| :------------------- | :---------- | :------------------------------------------- |
| estimated_arrival_time| datetime   | 预计到达时间                                 |
| actual_arrival_time  | datetime    | 实际到达时间                                 |
| transfer_reason      | varchar(50) | 调拨原因:USER_REQUEST-用户请求,INVENTORY_BALANCE-库存平衡 |
| suggestion_id        | bigint      | 关联的调拨建议ID                             |
| order_id             | bigint      | 关联的调拨单ID                               |
| receiver_user_id | bigint | 接收用户ID(用户请求/预约兑现时填写,库存平衡为空) |

**说明**：
- `transfer_reason`字段用于区分调拨来源
- `suggestion_id`字段用于关联主动调拨建议
- `order_id`字段用于关联调拨单,实现批量调拨管理
- 新增字段支持主动调拨功能的完整追踪

**表关系说明**：
```
tb_transfer_suggestion (调拨建议)
    ↓ order_id
tb_transfer_order (调拨单)
    ↓ id
tb_book_transfer (调拨记录,多条记录共享同一个order_id)
```
