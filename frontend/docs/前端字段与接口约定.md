# 前端字段与接口约定（中文展示 · 不改后端）

本文约定：**请求仍使用后端原有 JSON / Query 字段名**；**界面文案、枚举释义、结果解析**一律在前端完成。详细接口路径与参数仍以仓库根目录 **`API接口文档-前后端对接.md`** 为准。

---

## 1. 统一响应 `Result<T>`

| 后端字段 | 含义 | 前端处理 |
|----------|------|----------|
| `code` | 200 成功，400/404/500 失败 | `code !== 200` 时由 Axios 拦截器已 `ElMessage.error`，一般不再重复提示 |
| `message` | 提示语 | 可直接展示 |
| `data` | 业务载荷 | 按模块解析；**同时**可在折叠区保留 `JSON.stringify(data)` 供调试 |

| `reservationId` | 预约记录编号 | |
| `transferId` | 调拨记录编号 | |
| `copyId` | 图书副本编号 | |
| `sourceLibraryId` | 调拨来源馆编号 | |
| `targetLibraryId` | 目标馆编号 | |
| `taskId` | 任务编号 | 字符串 |
| `estimatedArrivalMinutes` | 预计到达（分钟） | |
| `recommendSeatLibraryId` | 推荐预约座位所在馆 | |
| `recommendSeatDate` | 推荐预约日期 | |

## 5. 调拨相关 `/api/transfer/*`

### 5.1 通用

列表类 `data` 常为：`total`、`page`、`size`、`records`。界面可展示「共 x 条、第 y 页」，表格列头用中文，**单元格内仍显示后端字段值**（如状态码英文）。

### 5.2 调拨记录状态（`BookTransfer.status` 等）

可在前端维护 `状态码 → 中文` 映射表，未命中则显示原文。

## 6. 座位 `/api/seat/*`

### 6.1 查询可用 `GET /api/seat/available`

| Query 键 | 界面中文 |
|----------|----------|
| `libraryId` | 图书馆编号 |
| `reservationDate` | 预约日期 |
| `startTime` / `endTime` | 开始 / 结束时间 |
| `areaId` | 区域编号（可选） |
| `hasPower` | 仅要有电源座位 |
| `autoAssign` | 是否自动分配一个座位 |

### 6.2 响应 `data`（`SeatAvailabilityResultDTO`）

| 后端字段 | 界面中文 |
|----------|----------|
| `availableTotal` / `total` | 可用座位数量 |
| `seats` | 座位列表 |
| `assignedSeat` | 系统推荐座位 |

`SeatInfoDTO`：`seatId`→座位编号；`areaId`→区域编号；`areaName`→区域名称；`floor`→楼层；`seatNo`→座位号；`hasPower`→是否有电源（1→有，0→无）。

### 6.3 创建预约 Body

| 后端字段 | 界面中文 |
|----------|----------|
| `userId` | 用户编号 |
| `seatId` | 座位编号 |
| `libraryId` | 图书馆编号 |
| `reservationDate` | 预约日期 |
| `startTime` / `endTime` | 时段 |
| `borrowId` | 关联借阅编号（可选） |
| `source` | 来源（可选） |

## 9. 前端实现约定

1. **表单 label**：一律用本表「界面中文」，**v-model 仍绑定后端字段名**（或通过 `reactive` 对象键名与后端一致）。  
2. **按钮文案**：用业务动作描述，**不把 URL 作为主标题**；必要时在 `title` 属性或次要说明里保留路径。  
3. **成功结果**：优先用 **`el-descriptions`** 展示「中文标签 + 值」；原始 JSON 放入 **`el-collapse`「原始数据」** 折叠。  
4. **列表**：`el-table` 列头中文，`prop` 仍对应后端字段。  
5. **新增映射**：只改前端 `src/utils/labels.ts`（或同类文件）及本 MD，**不改 Java**。
